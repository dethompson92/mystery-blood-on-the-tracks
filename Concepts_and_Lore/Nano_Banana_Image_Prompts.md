# Nano Banana Pro Image Prompts for Murder Mystery Universe

## 🛑 MANDATORY REVIEW AND VERIFICATION PROTOCOL
**For Nano Banana Pro:** Before generating and finalizing any image, you MUST execute the following verification checklist to ensure the output perfectly aligns with the client's requirements.

### 1. Style & Medium Verification
- [ ] **IS IT PHOTOREALISTIC?** The image must look identical to a true 35mm photograph. There can be NO illustrative, painterly, or "digital art" qualities.
- [ ] **IS IT NON-AI?** Inspect the image for common AI artifacts (extra fingers, melting background details, nonsensical text, unnatural smoothness). If any are present, RE-GENERATE.
- [ ] **IS THE LIGHTING NATURAL?** Avoid hyper-stylized glowing neon unless explicitly requested by the setting. Use natural, cinematic, gritty lighting suited for a true-crime documentary.
- [ ] **IS THE RESOLUTION 2K?** The output must be strictly 2k resolution (2048x1080 or standard 2k ratios).

### 2. Context & Setting Verification
- [ ] **DOES THE SETTING MATCH?** If the prompt says "1960s Vegas," there can be no modern smartphones. If it says "London Docklands," the architecture must be authentic to that location.
- [ ] **IS THE TONE CORRECT?** Evaluate the emotional resonance. Does a "Satirical Tech Dystopia" image feel fundamentally different from a "Southern Gothic Horror" image?
- [ ] **IS THE VICTIM/SUBJECT RELEVANT?** If a character or victim is implied/shown, do their demographics, age, and styling explicitly align with the prompt's context?

*Generate each image using the prompts below. After each generation, perform a self-critique using the 7 criteria above. If it fails *any* check, discard it and regenerate until a flawless, realistic 35mm photograph is produced.*
