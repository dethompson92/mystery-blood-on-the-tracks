IV. Technical Recovery Details

Messages were stored in the iOS SQLite database at /private/var/mobile/Library/SMS/sms.db. Deletion in iOS removes the row from the main message table but does not immediately overwrite the underlying pages. The messages were recovered from the SQLite Write-Ahead Log (WAL) file and from unallocated page space within the database file.

All recovered messages were verified using SHA-256 hash comparison against the full filesystem extraction image. No evidence of tampering or injection was found. The recovered content reflects the original messages as sent and received.
